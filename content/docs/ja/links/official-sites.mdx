# リンク集

QAに関する公式サイト・参考リソースへのリンク集です。

## ■JSTQB / ISTQB

### <a href="https://jstqb.jp/index.html" target="_blank" rel="noopener noreferrer">JSTQB（Japan Software Testing Qualifications Board）</a>

日本におけるソフトウェアテスト技術者資格認定の運営団体。ISTQB の日本支部として、FL・AL の試験を実施しています。

### <a href="https://istqb.org/" target="_blank" rel="noopener noreferrer">ISTQB（International Software Testing Qualifications Board）</a>

ソフトウェアテスト資格の国際標準を策定する団体。シラバスや用語集を無償公開しています。

## ■JISC

### <a href="https://www.jisc.go.jp/index.html" target="_blank" rel="noopener noreferrer">JISC（Japan Industrial Standards Committee）</a>

ISOの規格書（原本）は著作権で保護された有料の出版物であるため、公式サイト上でPDFなどが無料で全文公開されているわけではありません。  
しかし、こちらの日本産業標準調査会（JISC）のサイトで、簡単な利用者登録（無料）をするだけで、日本国内で「JIS X 番号」として完全に翻訳・規格化されたJIS規格の全文をブラウザ上で無料閲覧できます（※印刷やダウンロードは不可）。

### <a href="https://www.jisc.go.jp/app/jis/general/GnrJISSearch.html" target="_blank" rel="noopener noreferrer">JIS規格検索</a>

JIS規格の検索ページ。利用者登録後、キーワードやJIS番号で検索できます。

## ■テスト自動化ツール

### <a href="https://playwright.dev/" target="_blank" rel="noopener noreferrer">Playwright</a>

Microsoftが開発したオープンソースのブラウザ自動化ツール。クロスブラウザテストやエンドツーエンドテストに広く利用されています。

## ■CI/CD

### <a href="https://docs.github.com/ja/actions" target="_blank" rel="noopener noreferrer">GitHub Actions</a>

GitHubが提供するCI/CDプラットフォーム。コードのビルド、テスト、デプロイなどのワークフローを自動化できます。
